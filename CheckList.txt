- 공통 디자인
    - 라운드 코너는 하나의 값으로 통일 ✅
    - 텍스트는 제시한 UI와 동일하게 작성 ✅
    - 이미지, 아이콘 등은 SF Symbol 또는 Flaticon, Unsplash에서 최대한 유사한 리소스 사용(AI 생성 이미지 사용 가능) ✅
    - 컬러는 디지털 컬러 측정기 등을 사용하여 제시한 UI와 동일한 컬러 사용 ✅
    - Light Mode / Dark Mode 대응 ✅
    - 가로모드 / iPad, Mac 미대응 (iPhone 세로 모드만 구현) ✅
    - 텍스트 필드에 삭제 버튼 표시하기(편집중일 때만) ✅
    - UI는 최대한 스토리보드를 활용해서 구현하고, 스토리보드로 불가능한 부분은 코드로 구현 🟡
- 네비게이션
    - 좌측 1번 화면에서 Login 선택시 2번 화면 PUSH ✅
    - 중앙 2번 화면에서 Sign up 선택시 우측 3번 화면 Fullscreen 모달로 Present ✅
    - 우측 3번 화면에서 Login(Don't have an account? 오른쪽) 선택시 3번 화면 Dismiss ✅
    - 네비게이션 바는 숨김 ✅
- Get Started 화면
    - Get Started 버튼의 너비는 디바이스 너비에 따라 늘어나거나 줄어들어야 함 ✅
- Login 화면,
    - 화면이 표시되는 시점에 Email 필드에 포커스  ✅
    - Email 필드에서 Next 키 터치하면 Password 필드로 포커스 이동  ✅
    - Password 필드에서 Done 키 터치하면 Log in 버튼과 연결된 액션 실행  ✅
    - Log in 버튼을 터치하면 입력한 이메일과 비밀번호를 콘솔에 출력 ✅
    - SNS 로그인 버튼은 기능없이 배치만 구현 ✅
    - Forgot Password 버튼 선택하면 콘솔에 로그 출력(로그 문구는 자유) ✅
    - Email이 올바른 포멧✅으로 입력되었고 비밀번호가 아래와 같은 형식인 경우에만 Log In 버튼을 활성화
        - 길이 : 6 ~ 20 ✅
        - 대소문자 하나 이상 포함✅
        - 숫자 하나 이상 포함✅
        - 특수문자 하나 이상 포함✅
    - 텍스트 필드와 버튼의 너비는 디바이스 너비에 따라 늘어나거나 줄어들어야 함✅
- Sign Up 화면
    - 화면이 표시되는 시점에 Name 필드에 포커스✅
    - Name 필드에서 Next 키 터치하면 Email 필드로 포커스 이동✅
    - Email 필드에서 Next 키 터치하면 Password 필드로 포커스 이동✅
    - Password 필드에서 Done 키 터치하면 Log in 버튼과 연결된 액션 실행✅
    - Log in 버튼을 터치하면 입력한 이메일과 비밀번호를 콘솔에 출력✅
    - SNS 로그인 버튼은 기능없이 배치만 구현✅
    - Forgot Password 버튼 선택하면 콘솔에 로그 출력(로그 문구는 자유)✅
    - 이름이 2 ~ 5자 사이로 입력✅되었고, Email이 올바른 포멧으로 입력되었고, 비밀번호가 아래와 같은 형식인 경우에만 Log In 버튼을 활성화
        - 길이 : 6 ~ 20✅
        - 대소문자 하나 이상 포함✅
        - 숫자 하나 이상 포함✅
        - 특수문자 하나 이상 포함✅
